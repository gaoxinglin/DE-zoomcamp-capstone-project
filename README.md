# NZ Electricity Generation Pipeline

An end-to-end batch data pipeline built with **Bruin** that ingests NZ electricity
generation data from the Electricity Authority, unpivots wide-format CSVs using
BigQuery SQL, models generation trends by fuel type, and visualises NZ's energy
transition in Looker Studio.

## Stack
Bruin CLI · Bruin Cloud · BigQuery · GCS · Terraform · Looker Studio

## Why Bruin?
Bruin replaces Airflow + dbt + Great Expectations with a single tool.
One `bruin run` command executes ingestion, transformation, quality checks,
and orchestration — no Docker Compose, no separate CLI tools.
